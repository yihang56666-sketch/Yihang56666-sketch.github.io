项目简介

本项目实现了一个基于 SIFT + FLANN + RANSAC + Homography 的传统图像拼接（全景拼接）系统。以三张存在重叠区域的校园风景图作为输入，以中间帧为参考帧，将左右两张图变换投影到参考帧坐标系下，融合得到一张完整的全景图。

实现流程

1. SIFT 特征提取

对三张输入图像分别提取 SIFT（Scale-Invariant Feature Transform）特征点。SIFT 具有尺度不变性和旋转不变性，适用于存在视角差异的图像匹配场景。




三张图像上的 SIFT 特征点可视化

2. FLANN 特征匹配

使用 FLANN（Fast Library for Approximate Nearest Neighbors）匹配器进行快速近似最近邻搜索，配合 Lowe’s 比值检验（阈值 0.7）筛选高质量匹配对，剔除模糊匹配。


0023 ↔ 0024 特征匹配结果


0025 ↔ 0024 特征匹配结果

3. RANSAC 估计单应矩阵

利用匹配好的特征点对，通过 RANSAC（随机抽样一致）算法鲁棒地估计图像间的单应变换矩阵（Homography），同时剔除误匹配点。

4. 图像映射与融合

以中间帧 0024.jpg 为参考坐标系，将左右两图通过单应矩阵变换投影到参考帧空间。采用直接覆盖融合策略（非零像素覆盖），最后自动裁剪黑边。

拼接结果

最终的拼接效果相当自然，三张图像在重叠区域的对齐精度较高，接缝处过渡无明显断裂感。

拼接参数：输入 3 × 576×384 → 输出 955×463

总结

本实践完整走通了传统图像拼接的经典流程：

步骤 方法 作用
特征提取 SIFT 尺度/旋转不变的特征点检测
特征匹配 FLANN + Lowe’s 0.7 快速近似匹配 + 低质量筛除
变换估计 RANSAC + Homography 鲁棒估计透视变换关系
图像融合 直接覆盖 + 黑边裁剪 生成完整全景图

这种基于特征的传统方法在纹理丰富、重叠度适中的场景下效果非常稳定，同时也是理解现代深度学习特征匹配方法（SuperPoint、SuperGlue 等）的重要基础。
