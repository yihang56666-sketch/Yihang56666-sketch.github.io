项目概述

这是一个面向嵌入式硬件工程师的 AI 辅助工具集，让 Claude Code 等 AI 编程助手能够理解芯片数据手册、开发板原理图，并提供外设配置、引脚分析、CubeMX 辅助等能力。项目包含三个部分：

  • chip skill — 核心 Python 工具，提供命令行接口
  • chip-mcp — MCP 协议服务器，供 AI 工具直接调用
  • 硬件分析报告 — 对 MSP430G2、STM32F407VET6 等芯片的深度分析文档

MCP 服务器

chip-mcp/server.py 将 Chip Skill 包装为标准 MCP（Model Context Protocol）服务器，使得 Cline、Cursor 等 AI 编码工具可以直接调用其功能。MCP 协议定义的工具有：

工具名 功能
search_datasheet 搜索芯片数据手册
search_schematic 搜索板子原理图
analyze_board 分析板子硬件特性
cubemx_assist CubeMX 配置辅助
pinout_analysis 引脚分配分析
peripheral_guide 外设使用指南
task_solution 硬件任务解决方案

硬件分析报告

项目中还包含两份深入的芯片分析报告：

MSP430G2 LaunchPad

德州仪器 (TI) 的超低功耗入门级开发板：

  • MCU: MSP430G2553 (16位 RISC, 16MHz)
  • 存储: 16KB Flash, 512B RAM
  • 外设: 10位 ADC, 2×定时器, UART/I2C/SPI
  • 特色: 超低功耗（uA 级），适合电池供电应用

STM32F407VET6

意法半导体高性能 Cortex-M4 微控制器：

  • CPU: ARM Cortex-M4 @168MHz, 单精度 FPU
  • 存储: 512KB Flash, 192KB SRAM
  • 外设: 3×12位 ADC, 2×DAC, DMA, USB OTG, 以太网
  • 特色: DSP 指令集，适合数字信号处理

总结

这个工具集的核心思路是：将硬件知识结构化，让 AI 能够理解和运用。传统嵌入式开发中，工程师需要反复查阅数据手册、原理图、外设参考手册，而通过这套工具，AI 助手可以在对话中直接获取这些信息并辅助开发——从选型分析、外设配置到代码生成，形成完整的辅助闭环。

项目基于 MIT 协议开源，支持自定义开发板数据库，方便团队接入自有硬件平台。
